I posted the bare bones of this build some time back. But now I've fleshed out the build.

Str14
Dex15
Con14
Wis14
Int10
Cha10
All Stat increases into Dex

(1)Barb1:Blooded, Thug
(2-3)Barb1/Monk2:Weapon Finesse
(4-6)Barb1/Monk5:Imp Initiative
(7-9)Barb1/Monk8:Weapon Focus(unarmed)
(10-12)Barb1/Monk11:Blindfight
(13-15)Barb1/Monk14:Toughness
(16-18)Barb1/Monk17:Improved Critical(unarmed)
(19-21)Barb1/Monk20:Gr Dex
(22-24)Barb1/Monk23:Gr Dex 2
(25-27)Barb1/Monk25:Gr Dex 3, Armour Skin
(28-30)Barb1/Monk29:Gr Dex 4
(31-33)Barb1/Monk32:Gr Dex 5, Imp Spell Resistance
(34-36)Barb1/Monk35:Blinding Speed, Imp Spell Resistance
(37-39)Barb2/Monk37:Superior Initiative
(40)Barb2/Monk38

Theme is to move faster than anyone possible without any additional equipment, also the initiative is also higher than most other dex based builds. Win initiative, hit and run. High dex also aids in using ranged weapons, so it is possible to attack using a bow and run away since it is impossible for anyone else to catch up.
