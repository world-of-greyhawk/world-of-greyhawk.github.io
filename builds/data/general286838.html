I play Dragon Mountian. It is nwn1 world with a custom Prc implementation and CEP2. We have over 1000 area, 5 cities, active guilds, and a slight RP flair. It is an action based server mostly, but the active Dms run rp based storylines.

we have eliminated several prc classes for balance, but mostly to eliminate lag or other technical issues. this is a very low lag server. We continue to look at opening other classes as well.

We have a Custom quest system and a 'token' system that allows the creation of custom items. You have multiple quests at each level beyond 5th and bosses at each level. Most levels have more than one boss so multiple runs to 40 do not mean you will face the same boss 2x.

We favor expert builds. it is a very hard world and will require a great build to solo, but it is a party based server so people will often switch chars to party with you. Many people have been on the world more than 3 years.


If you like the epic builds highlighted on these forums, we have a 40+ level castle area where most of these builds have been implemented for you to fight. A great way to test what you may consider the 'ultimate' build. I think you will find any build fully tested by the mountian. Can you climb to the top?

We just implemented a Capture the flag full PvP system as well for higher level chars.
