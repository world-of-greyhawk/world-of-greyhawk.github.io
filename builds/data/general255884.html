Just a few hours after the Presale Toolset was available for download I started working on a CBC for NWN2.

Because of so many differences between NWN and NWN2 and because I wasn't really happy with many design decision I made when making the NWN1 CBC I've decided to create a new CBC2 from scratch instead of porting the NWN1 CBC to NWN2. Therefore (and because NWN2 isn't published yet and nearly all my knowledge comes from the presale toolset 2da files) at the moment the CBC2 contains much less functionality than the old CBC1. That will definitely change once I have NWN2 in my hands.

At first I just wanted to post some screenshots but then I thought that I could also upload an alpha version. After all the module builders have the toolset to play with but what about the character builders?

CharacterBuildCalculator for NWN2 (alpha version)

At the moment it waits for approval but it should be available for download in the next hours.

The version is alpha (or beta maybe) but most basic things should work fine.

Any feedback is appreciated.

Somewhat late but...

The first versions of CBC2 had quite a few bugs but most of them should be fixed now.

Keep in mind however that the default setting for skill checks is 'simple skill checks' and that means that only maximal possible skill ranks are checked. So a rogue(7) with Dodge/Mobilty can start to take SD levels but a fighter(7) with Dodge/Mobility can't. Only if you set 'skill checks' to 'full skill checks' you actually have to buy skill ranks to meet skill prerequisites.
