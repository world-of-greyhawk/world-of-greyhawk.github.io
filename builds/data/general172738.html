If someone could explain it to me from after the point I have logged into "Multiplayer" at the main menu, that would be most appreciated. What you need to do; is download the module from the link provided.

1) It is usually a zip or rar file, uncompress this using your favourite decompression tool (I use winrar, which works for zip and rar: http://www.rarlab.com )
2) You place the module that is decompressed in your modules directory (so if your nwn is installed in the default location on the c: drive; this would be c:\NeverwinderNights/NWN/modules).
3) You then load up neverwinternights and select the NEW button when the game starts. You should see a list of installed NWN official campaigns (and if you have the latest patch you should see -ugh- a premium button as well).
4) Click on the last button, labelled 'Other'. The next screen will include (provided you have copied the module to the right place) a link for the novice to epic module.
5) There you have it . The same process will work for the single-player modules also available on the vault.
