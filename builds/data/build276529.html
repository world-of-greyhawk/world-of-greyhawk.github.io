This is my first post of a build so please bear with me. It is a blend of my two favored classes, Blackguard and Rogue. There are other builds that use these classes but none with this level breakdown that I saw. It was an attempt to get the most out of the skills of each class, with rogue skills such as PP and Set Traps neglected. Set Traps could be useful for extra damage though. Also, the world I played this character on had a large number of undead and I was becoming frustrated with the damage output of my character, hence Divine Might.

Known Weaknesses of build:
Only 269 skillpoints- For a rogue this seems low. It's due to the 10 intelligence, of course.
Only 28 Dex- This negatively affects the AC and AB of the character but Divine Shield should counter that for AC.
Only 18 Char- In a build using Divine feats this is a bit low but with the right Charisma gear this can be overcome.
Low Damage Base- With only 14 strength the build isn't going to put out a lot of base damage but it does have a 16d6 sneak attack.
No Focus/Epic Focus HS/MS- On a high magic PvP server you won't much be able to depend on your Stealth to land sneak attacks unless your target hasn't built up their Spot/Listen. It's still passable, though.
No dual wield- That speaks for itself.
Low Saves- Adding Charisma gear will help some but the build doesn't have Slippery Mind.

Gear
You'll want to aim for charisma and dex gear for this character. You'll probably want to get usage out of UMD and get Improved Invis scrolls and Bless Weapon/Darkfire/Divine Favor/Holy Sword, Etc. Strength could be helpful in boosting base damage. Also, anything that helps raise Will Save would be nice.

This build could be done with an elf as well. You would lose 43 skill points, 40hp, and one feat but you would gain 1AB and 1AC. The character is already starved of skill points being a rogue in my opinion, though.

Any suggestions on this build?
