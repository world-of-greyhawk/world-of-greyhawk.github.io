Commentary This build is intended to grant the maximum number of Wizard spells per day, as well as the highest DC possible without resorting to an egregious loss of caster levels (i.e. Wiz29/Brd1/DD10 for an addition +2 to INT). Likewise, this Wizard has an extremely high Initiative modifier, which will be upwards of +18 with magical bonuses to Dexterity, and is quite handly in spellbattles, pvp, or when the character is suprised in a PvM module.

The majority of Epic Feats available to Epic spellcasters either tend to be useless or can easily be replicated via Magic item properties. Thus this Wizard build operates on the notion of the player having access to at least some relatively decent equipment properties. As for the more unorthodox or often nerfed item properties (i.e. Boots of Haste, or a magic item that casts Epic Spells), simply follow the guidelines below to augment the character build to the individual server situations:

If the intended server does not allow permanent Haste items, then either use Extended Haste spells, craft Wands of Haste, or simply remove Superior Initiative and two Great Intellect feats and replace them with Automatic Quicken Spell I, II, and III.

If the intended server does not allow magic items that duplicate the effects of Epic Spells, then simply remove Superior Initiative, and replace it with Epic Spell: Epic Warding instead. Of all the Epic Spells, this is the only one which is truly worthwhile.

If the intended server does not allow Armor class bonuses beyond +5, then likewise remove an Epic Feat in place of Epic Spell: Epic Mage Armor, Although if you are a 40th level character limited to below 20th level armor bonuses, then perhaps I would consider a server switch...
