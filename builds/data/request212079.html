The first build I tried was a Blackguard/Rogue/SD which seems somewhat similar to what you're trying for with all the stacked sneaks. I really DON'T recommend blackguard for a rogue type. Rogues don't get many bonus feats which you can take for stuff like KD, Armor Skin, etc and Blackguard does NOT help much with that at all. If only they'd put in a CoT-like class for evil PCs. You'll probably really really struggle to get all the feats you'd like when you'd like (especially if you plan on getting Power Attack/Cleave for Blackguard levels pre-epic and Exotic AND dual wielding) with that class combo as an elf. I mean you'll make it but it'll be during the post-epic levels. If you're going assassin-heavy, you might have trouble getting epic dodge.
