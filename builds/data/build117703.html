I've been working on a Pal/WM/COT build as well and found this one, but this one has errors. I just wanted to point them out in case someone else attempts to build this.

1. Whirlwind attack is a pre-req for WM and this build has it taken at WM5 or character level 18. By using the extra feat for a human you can overcome this by shifting your feats up some.

2. Biggest error is that epic COT gets bonus feats every 3 levels past 10 and not 2! That severly changes the build. You can't get all the feats stated that way! Gray,

Rather than ask people to point out errors, why don't you go build this character yourself, post the correct leveling guide, and then ask for feedback on a buildable character?
