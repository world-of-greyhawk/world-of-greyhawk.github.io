I am endeavoring to construct mnemonic memory circuits, using stone knives and bear skins. You might want to adjust the skillpoints a bit. For Disable Traps, you only need to reach 38 to disable Epic traps (and only 26 to recover all Deadly traps). For Set traps, you can set all Deadly ones with 25. I don't think there are many Epic traps in Aielund, so you might want to aim for Deadly levels. That will save you some points on Search as well. I would put some of those points towards UMD - for scrolls, wands, etc.

Last, you might want to consider switching to Fighter instead of CoT for WS/EWS. Monk 20/Rogue 11/Fighter 9 (the 11th Rogue level is so you can go for Epic Dodge as early as possible but have one more skill dump afterward). You will need to balance fighter and rogue levels to avoid an XP penalty. You could pick up an extra bonus feat by going Rogue 10/Fighter 10, but then you lose the skill dump or delay Epic Dodge.
