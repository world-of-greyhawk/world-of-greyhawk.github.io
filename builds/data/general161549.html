For the most above, you might want to use magic-based class. I would perfer wizard. As you said, the world is low in magic. So the wizard would have advantages here, they are powerful when the other classes don't have defenses against them. If it was heavy magic world, magic-based characters would be at disadvantage.

Anyway, it would be recommended that you do some combination of Wizard for high lore (especially that they have spell that boost their lore skills), Rogue for more skill points and they use spot and listen skills I think. If you want to have third class, it would be recommended that you add in Palemaster because they have immunity against criticals. If you were to encounter any characters that uses criticals, it would come to be useful. But if the level progression is slow... it might make some difference because you can get it at fourth level if you meet all requirements. Except that you would not be able to get immunity to criticals until you have 10 levels in palemaster.

I don't have level progression for what I suggested. But I have done a build like this one time long ago, but it has been while since I have last used it.

The reason for the combo of Wizard/Rogue/ and possibly Palemaster is that they have ability to sneak around and use the magics. They can cast the magics, and sneak around again after they are seen. It is good tactics, and sometimes it annoys the others... lol. And if I am correct, the world would not have any items that grants True Seeing, is that correct? If that is true, then this build would be strong in that world. Palemaster is only taken to give you immunity to the criticals.

Anyway, the suggestion above is based on my preferences.

Others would have better suggestions than I have. But at least you have option.
