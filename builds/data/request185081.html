Here's a build that takes the above classes and gets all the Epic shapes, although it doesn't really focus on any shape in particular.

When in Human form, the character should be a fairly strong caster, but his offense will be fairly weak due to his using a ranged weapon.

Note that you get Infinite Elder Elemental Shape at Druid lvl 26, so you have pretty much all the shifting power you could want, with the exception being that you don't get the Epic Humanoid forms. It's not really a big deal, although the Epic Kobold Commmando is kind of fun if you like to play Rogues.

Although this character will be able to use a sling, if you want to use a bow you will need to buy the Martial Weapon Proficiency. As for Kama, it doesn't make much sense for a Wis-based build.

Of course, it is possible to do a Shifter build without being Wis-based (focusing on elemental, humanoid, or Undead forms) but you wouldn't be able to get Outsider, Construct, or Dragon Shape, as these all require a high Wisdom score.
