I really liked the look of this build so I decided to test it out. Two things popped up.
1) In the character build process, you need to take Epic Weapon Focus before Epic Weapon Spec. This can be done by taking the last fighter level in this build a little later.
2) The other thing is that it doesn't appear that devastating critical actually works with the longbow. While testing this build, I must've had a dozen crits and none of them generated a devastating critical roll. (The monsters were not crit immune)

Can anyone confirm that Devastating Criticals still don't work with longbows? I just did a little research.
It looks like devastating criticals for longbows is suppose to appear in the v1.63 patch.
When that happens, this will be an awesome build.
