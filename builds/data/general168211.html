is it possible to get a dex based monk devestational crit? and if so how? Probably possible. You only need to get your strength to 25 for Dev Crit.
Start with Strength as high as possible and then bump it every 4 levels. Perhaps take one or two Great Strength feats. Then you could take some Great Dexterity feats to build up dex.

Note that you'll have to fit in the Dev Crit feats (Power Attack, Great Cleave, Imp. Crit, Overwhelming Crit, and Dev Crit) in addition to using Great Str/Dex feats so it might stretch you pretty thin.

Try building one yourself in a training/off-line mod, then post the build here for critique.

Kaliban.
