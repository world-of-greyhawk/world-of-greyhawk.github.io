I don't think this is possible. You have HS and WM as PrC classes, so the 3rd class must be a base class. You get 7 general feats plus 2 HS bonus feats, that's 9 Great Dex. No base class gets Great Dex as a bonus feat, so I don't think you can get the 10th one. You would need some levels of AA or Assassin or something to do that, which would mean dropping either HS or WM.
